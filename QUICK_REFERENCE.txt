┌──────────────────────────────────────────────────────────────┐
│  FASTEST WAY TO DEPLOY (2 minutes)                          │
└──────────────────────────────────────────────────────────────┘

1. Go to → https://vercel.com
2. Sign up (free)
3. Click "Add New" → "Project"
4. Drag & drop the "heybuddydesign-project" folder
5. Click "Deploy"
6. Done! 🎉

Your URL: https://heybuddydesign.vercel.app

┌──────────────────────────────────────────────────────────────┐
│  DEPLOY VIA CLI                                              │
└──────────────────────────────────────────────────────────────┘

# Install Vercel CLI
npm install -g vercel

# Login
vercel login

# Deploy preview
vercel

# Deploy production
vercel --prod

# Or use the script
./deploy.sh

┌──────────────────────────────────────────────────────────────┐
│  CUSTOM DOMAIN                                               │
└──────────────────────────────────────────────────────────────┘

1. Buy domain (Namecheap, GoDaddy, Cloudflare)
2. In Vercel: Settings → Domains → Add
3. Update DNS:
   Type: CNAME
   Name: @
   Value: cname.vercel-dns.com
4. Wait ~30 minutes
5. Done! Free SSL included
